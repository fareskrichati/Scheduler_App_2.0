# Pulse Planner

Pulse Planner is a lightweight scheduler app that runs entirely in the browser with no build step or backend required.

## Features

- Week navigation with a selected-day timeline
- Create, edit, delete, and complete events
- Agenda filters for category and search
- Local persistence through `localStorage`
- Responsive layout for desktop and mobile

## Run it

Open [index.html](./index.html) directly in a browser. The app does not need a build step, package install, or backend server.
